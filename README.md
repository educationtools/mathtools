# MathTools

## Veröffentlichen
1. Statische Dateien erzeugen

    npm run build
    
2. Dateien veröffentlichen (Repos müssen in mathtools/mathtools.github.io vorhanden sein)

    ./deploy.sh
    
3. Ergebnis bewundern 

    https://mathtools.github.io
